VerifyLens - Deepfake Video Detection Website

VerifyLens is a web-based application designed to detect deepfake videos using cutting-edge AI techniques. Built using TensorFlow, Keras, and the Xception model for feature extraction, the website leverages Flask for the backend and HTML, CSS, and JavaScript for the frontend. The project utilizes datasets from Kaggle Celeb-DF for training and evaluation.

📝 Features
Upload videos and detect whether they are real or deepfake.
Utilizes the Xception model for feature extraction from video frames.
Frontend designed with HTML, CSS, and JavaScript for an intuitive user experience.
Backend powered by Flask, handling video uploads and model inference.
Real-time prediction display on the website.
